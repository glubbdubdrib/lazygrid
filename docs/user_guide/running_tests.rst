Running tests
=============

You can run all unittests from command line after having
downloaded the source code from
`GitHub <https://github.com/glubbdubdrib/lazygrid>`__:

.. code:: bash

    $ git clone https://github.com/glubbdubdrib/lazygrid.git
    $ cd ./lazygrid

You can use either python:

.. code:: bash

    $ python -m unittest discover

or coverage:

.. code:: bash

    $ coverage run -m unittest discover
